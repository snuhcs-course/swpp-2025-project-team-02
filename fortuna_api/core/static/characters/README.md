# Character Images for Fortune Telling

This directory contains character images for each of the Five Elements (오행).

## File Structure
- `wood.png` - 목 (Wood/Tree) element character
- `fire.png` - 화 (Fire) element character
- `earth.png` - 토 (Earth/Soil) element character
- `metal.png` - 금 (Metal) element character
- `water.png` - 수 (Water) element character

## Usage
These images are uploaded to OpenAI Files API and referenced when generating fortune comics.
The character used is determined by the user's day stem element (일간 오행).

## Image Requirements
- Format: PNG with transparent background
- Style: Cute chibi/kawaii character
- Size: Recommended 512x512 or higher
- Content: Character representing each element's personality and visual traits
